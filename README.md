# tikDM - TikTok Direct Message Chrome Extension

tikDM is a Chrome extension that allows you to send direct messages on TikTok from your browser.

## Features

* Send direct messages to any TikTok user.
* Easy to use interface.
* Open source.

## Installation

1. Download the `tikDM_chrom_extension.zip` file from the [releases page](https://github.com/wk23aau/tikDM/releases).
2. Unzip the file.
3. Open Chrome and go to `chrome://extensions`.
4. Enable "Developer mode".
5. Click "Load unpacked" and select the unzipped folder.
